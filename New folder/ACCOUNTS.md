# 🔐 حسابات التجربة - Focus App

## المشروع الآن متصل بـ API! ✅

### الخوادم الشغالة:
- **Frontend (Vite):** http://localhost:5174
- **Backend API (json-server):** http://localhost:3001

---

## 📋 الحسابات المتاحة للتجريب

### 🎓 حساب طالب
```
البريد الإلكتروني: student@focus.com
كلمة المرور: 123456
```

### 👨‍🏫 حساب مدرس
```
البريد الإلكتروني: teacher@focus.com
كلمة المرور: 123456
```

---

## 🚀 خطوات التجريب

1. افتح المتصفح على: http://localhost:5174
2. اضغط على "تسجيل الدخول"
3. اختر نوع الحساب (طالب أو مدرس)
4. أدخل البيانات من الأعلى
5. استمتع! 🎉

---

## 📝 ملاحظات مهمة

### الصفحات المتاحة حسب الحساب:

**حساب الطالب:**
- `/dashboard` - لوحة تحكم الطالب
- `/subject/:id` - صفحة المادة
- `/course-details` - واجهة الدرس
- `/quiz` - الاختبار
- `/quiz-results` - نتائج الاختبار
- `/ai-chat` - الدردشة مع الذكاء الاصطناعي
- `/remediation` - خطة العلاج
- `/profile` - الملف الشخصي
- `/settings` - الإعدادات ⭐ (جديد)

**حساب المدرس:**
- `/teacher-dashboard` - لوحة تحكم المعلم
- `/profile` - الملف الشخصي
- `/settings` - الإعدادات ⭐ (جديد)

---

## 🔧 إيقاف الخوادم

في PowerShell اضغط `Ctrl+C` في Terminal الخاص بكل خادم

جميع البيانات محفوظة في `db.json` ويتم تحديثها مباشرة عند التعديل في Profile أو Settings.
